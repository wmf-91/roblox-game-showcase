# Roblox Game Project
Title of game upon publish : Launch For Brainrots!
Game link : https://www.roblox.com/games/109960992086299/test1

A Roblox game I developed in Roblox Studio as a solo developer including bug fixes, codings, modelling and interface designs.
This game is currently in development and will be expected to release by the end of June 2026. As of now, I can only provide screenshots and codes used to develop the game
to show you brief examples on how the game will look like and its mechanics.

Please refer to the Roblox Portfolio folder that contains screenshots and sample codes used in developing the game.

## Features
- Platform System
- Shop system
- Data saving and loading system

## Screenshots
Cannon System
- This game revolves around a cannon mechanic as its primary function where players will use the cannon to launch through walls depending on their firepower and power meter as shown
  in the screenshot.

Chasing System
- Upon picking up items, player will enter a state where the wall will chase them until the safe zone and will die upon contact and lose the item.

Map View
- A screenshot showing the game's map

Platform System
- A picture showing where items can be placed at player's base where it includes levelling and money collecting mechanics.

Shop System
- A shop system where player will purchase using Roblox's currency to receive paid exclusive stuffs.

Training System
- Player can train their firepower using weights to be able to reach much further walls

## Sample codes

Coreserver Script
- This script is responsible for base assignments, data saving and data loading of player upon joining or leaving

Datamanager Script
- This script is responsible in managing player's data

Platform Manager Script
- This script acts as a blueprint for the platform's mechanics involving the items placed on them.
